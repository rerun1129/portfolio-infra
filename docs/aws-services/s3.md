# S3 (Simple Storage Service)

## 개요

포트폴리오 서비스에서 사용하는 오브젝트 스토리지.

## 관리 방식

**Terraform 외부에서 수동 관리** (AWS 콘솔에서 직접 생성/설정)

### 이유
- S3는 켜고 끄는 개념이 없어 `terraform destroy` 대상이 아님
- 버킷 삭제 시 내부 데이터 전부 손실
- 1GB 이하 사용 시 월 $0.025(약 34원) 수준으로 비용 무시 가능

### Terraform에서 참조하는 방법
다른 리소스(EC2, Lambda 등)에서 S3 버킷 이름이 필요할 때 변수로 참조:

```hcl
variable "s3_bucket_name" {
  description = "수동 생성된 S3 버킷 이름"
  type        = string
}
```

## 리전

ap-northeast-2 (서울)

## 요금 (서울 리전, S3 Standard 기준)

| 항목 | 단가 |
|------|------|
| 스토리지 | $0.025/GB/월 |
| GET 요청 | $0.00035/1,000건 |
| PUT 요청 | $0.0045/1,000건 |
| 외부 데이터 전송 | 100GB/월 무료 |
